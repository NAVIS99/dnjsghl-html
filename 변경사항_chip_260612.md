# 칩 클래스 변경사항 (developer handoff) — 2026-06-12

> 비교 기준: 현재 개발자가 작업 중인 이전 버전(`.label-chip` 기준) → 최신 버전(`.chip` + 멀티클래스)

## 1. `.label-chip` → `.chip` + 사이즈 클래스로 분리

기존 `.label-chip` 한 클래스에 몰려있던 속성을, 베이스(`.chip`)와 사이즈 모디파이어(`small`/`medium2`)로 분리했습니다.

**변경 전**
```css
.label-chip {
  display: flex; height: 2.25rem; padding: 0 1rem;
  justify-content: center; align-items: center; gap: 0.25rem;
  border-radius: 31.25rem; border: none;
  font-size: 0.875rem; font-weight: 600; letter-spacing: -0.007rem;
  cursor: pointer; transition: background 0.1s, color 0.1s;
}
```

**변경 후**
```css
.chip {
  display: flex; justify-content: center; align-items: center;
  border-radius: 31.25rem; border: none; letter-spacing: -0.007rem;
  cursor: pointer; transition: background 0.1s, color 0.1s;
}
.chip.small   { height: 2.25rem; padding: 0 1rem; gap: 0.25rem; font-size: 0.875rem; font-weight: 600; }
.chip.medium2 { height: 2.5rem;  padding: 0 1rem; gap: 0.25rem; font-size: 0.9375rem; font-weight: 600; }
```

- `.chip.small` = 기존 `.label-chip` 사이즈(height 2.25rem) — 기존 칩은 모두 이 사이즈
- `.chip.medium2` = 신규 추가 (height 2.5rem) — 추후 medium 사이즈 칩 필요할 때 사용
- `.chip`(베이스)에는 사이즈 무관 공통 속성만 남음: `display`, `justify-content`, `align-items`, `border-radius`, `border`, `letter-spacing`, `cursor`, `transition`

## 2. 색상/상태 클래스명 변경

- `.label-chip.default` → `.chip.default`
- `.label-chip.default:hover` → `.chip.default:hover`
- `.label-chip.selected` → `.chip.selected`
- `.label-chip.disabled` → `.chip.disabled`
- 동작/색상 값은 동일, 사이즈와 무관하게 공통 적용

## 3. 컨테이너 클래스명 + gap 값 변경

- `.label-chips` → `.chip-group`
- gap 값 변경: `0.375rem` → `0.5rem 0.375rem` (row-gap 0.5rem, column-gap 0.375rem)

```css
/* 변경 전 */
.label-chips { display: flex; align-items: center; gap: 0.375rem; flex-wrap: wrap; align-self: stretch; }

/* 변경 후 */
.chip-group { display: flex; align-items: center; gap: 0.5rem 0.375rem; flex-wrap: wrap; align-self: stretch; }
```

## 클래스 치환 표 (마크업 적용)

| 항목 | 변경 전 | 변경 후 |
|---|---|---|
| 컨테이너 | `class="label-chips"` | `class="chip-group"` |
| 칩 (기본 상태) | `class="label-chip default"` | `class="chip default small"` |
| 칩 (선택됨) | `class="label-chip selected"` | `class="chip selected small"` |
| 칩 (비활성) | `class="label-chip disabled"` | `class="chip disabled small"` |

> `small`은 별도 클래스 토큰으로, `chip` 뒤에 공백으로 구분해서 추가해야 합니다. (`chip-small`처럼 한 단어로 붙이는 게 아님)
> 클래스 순서는 `chip` → (스타일/상태) → (사이즈) 순으로, 버튼(`btn-solid-black-large` = 스타일→컬러→사이즈) 컨벤션과 동일한 순서로 통일했습니다.

## 영향도

- JS 로직은 id/key 기반이라 영향 없음, 클래스명만 교체
- `.label-chip`, `.label-chips` 잔여 참조 없음 (전체 grep 확인 완료)

## 개발자 작업 필요사항

- 이전 버전 마크업으로 작업 중이라면, 위 표대로 클래스명을 전체 교체해주세요.
- 컨테이너의 `gap` 값이 바뀌었으므로(`0.375rem` → `0.5rem 0.375rem`), 칩이 줄바꿈되는 화면에서는 행간 간격이 약간 넓어집니다.
